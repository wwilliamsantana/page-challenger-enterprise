Perguntas Frequentes

O que é o Page Challenger Enterprise?

O Page Challenger Enterprise é uma plataforma para desafios de front-end design engineering.

Como posso participar dos desafios?

Para participar, basta se inscrever na plataforma e escolher um dos desafios disponíveis.

Quais são as tecnologias permitidas?

As tecnologias permitidas são HTML, CSS e JavaScript.

Como a pontuação é calculada?

A pontuação é baseada na qualidade do código, design e usabilidade da solução.

Sobre a ONG
