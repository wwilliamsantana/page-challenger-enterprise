Sobre o Projeto

Este projeto é uma iniciativa para a disciplina de Front-End Design Engineering. O objetivo é criar um site para a ONG "Turma do Bem", focando em um design moderno e atraente, utilizando apenas HTML e CSS para a primeira entrega.

A ideia

A ideia principal é desenvolver uma presença online para a ONG que seja informativa e acolhedora. Queremos que os visitantes entendam a missão da "Turma do Bem" e se sintam compelidos a participar, seja como voluntários ou doadores.

Tecnologias

  • HTML5
  • CSS3

Roadmap

Sprint 1: Base do Front-End

Criação das páginas estáticas (Início, Sobre, FAQ, Contato, Integrantes) com HTML e CSS. Foco no design para desktop.

Sprint 2: Interatividade

Adição de JavaScript para criar interações e melhorar a experiência do usuário.

Sprint 3: Responsividade

Tornar o site totalmente responsivo para dispositivos móveis e tablets.

Sobre a ONG

Nossa missão é fornecer cuidados odontológicos e apoio psicológico para comunidades carentes, promovendo saúde e bem-estar.
